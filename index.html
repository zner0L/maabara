Error Propagation made easy - maabara¶

Nothing is certain but necessity of error propagation

Maabara allows you to compute and document error propagation fast and pythonic through Latex.

  • Pypi Website
  • Github Project
  • Blog

Maabara follows semantic versioning. It is a swahili word meaning »laboratory«.
